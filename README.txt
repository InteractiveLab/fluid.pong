FLUID PONG
Multi-touch pong in Scala/Processing based on MSA Fluid and Box2D.
Copyright (c) Valentin Simonov, Interactive Lab. Distributed under MIT license.

It is advised to use IntelliJ IDEA and Scala 2.8.
Check out this post on how to use Processing with Scala: http://va.lent.in/blog/2011/03/04/scala-processing/.

Dependencies:
* jbox2d 2.1.2
* libTUIO
* MSAFluid
* slf4j

To run the app make sure that you have JOGL native libraries in class path.
